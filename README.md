# DoLearn Backend

Express API workspace for the DoLearn backend.

Planned stack:

- Express.js
- TypeScript
- PostgreSQL
- Prisma
- Zod validation

## Local Setup

```bash
npm install
cp .env.example .env
npm run dev
```

The API starts on `http://localhost:4000` by default.

## Current Status

This folder is a starter scaffold. The frontend still uses mock data and browser local storage until the backend API is implemented.
